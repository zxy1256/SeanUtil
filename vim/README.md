CheatSheet
------------

Buffer
========

    :w Save buffer to file
    :bd Close buffer
    :bn Next buffer
    :bp Previous buffer
    :b<#n> Jump to buffer #n

Program
=========

    :q Quit vim


Config
-------

 1. Use the vimrc in this folder by adding the folling lines to `~/.vimrc`

    ```
    source ~/Code/Github/Setup/vim/vimrc
    ```

 2. Install plugins
  - Install pathogen.vim
  - Install l9
  - Install NERDTree
  - Install FuzzyFinder
